## ✔ 주요 기능

- 가맹점관리
  - 가맹점 경고 기능

- 캘린더관리
  - 부서별 캘린더 분류


- 📋 전자결재
  - 결재 양식을 선택하여 기안문 상신
  - 결재자 수에 따른 결재라인 생성
  - 기안문 승인 및 반려
  - 결재 순서에 따른 메뉴 구분
  - 전자결재 문서 PDF 및 인쇄 기능
